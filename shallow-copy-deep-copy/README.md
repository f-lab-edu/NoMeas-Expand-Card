# 과제

```js
const a = { merong: 2 }
const b = a
```

이 코드에서 a는 참조 타입인 Object이기 때문에 b에 a를 할당하더라도
원시 타입들처럼 복사가 되지 않는다는 사실! 이를 얕은 복사라고 불러요.

그런데, 이를 복사할 수 있는 JavaScript 표준 API가 있어요.  
첫 번째 과제, 그 API가 무엇인지 알아오기입니다.

그런데 그 API는 문제점이 있는데요. 바로 얕은 복사밖에 안된다는 문제예요.  
얕은 복사란 무엇일까요? 그것을 알아오는 것이 두 번째 과제입니다.

그 다음 마지막 과제로, 이 얕은 복사 문제를 해결할 수 있는 깊은 객체 복사 함수를 만드는 것입니다.
최대한 다양한 자료형의 값을 복사할 수 있게 해주세요. (Map, Set, Date, RegExp 등)
Vitest를 이용해 테스트 코드를 작성해주세요.
